# Aishling Murray Doula Site

Simple static HTML site for GitHub Pages.

## Deploy on GitHub Pages

1. Push this repo to GitHub with `main` as the default branch.
2. In GitHub, go to `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `GitHub Actions`.
4. Push any commit to `main` to trigger deployment.

The workflow file is at `.github/workflows/pages.yml` and deploys `index.html` as a static site.
